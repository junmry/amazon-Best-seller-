## 🚀 本地访问方式

### 方式一：Python HTTP服务器
```bash
cd trading-learning-site
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 方式二：Node.js http-server
```bash
npm install -g http-server
cd trading-learning-site
http-server -p 8080
```

### 方式三：VS Code Live Server插件
右键 index.html → Open with Live Server

## 📚 课程内容

### 阶段一：零基础入门
1. **股票是什么？** - 面向完全小白的股票基础概念
2. **K线基础入门** - K线构成、形态、组合
3. **技术指标入门** - MA、MACD、KDJ、WR、BOLL
4. **股票交易基础常识** - 交易规则、开户、风险控制

### 阶段二：投资哲学
5. **股票大作手回忆录** - 杰西·利维摩尔的交易智慧
6. **投资心理学基础** - 情绪管理、认知偏差

### 阶段三：技术分析
7. **专业投机原理** - 维克多·斯波朗迪的三大原则、123法则
8. **短线交易秘诀** - 拉瑞·威廉姆斯的威廉指标、波动率交易

### 阶段四：基本面分析
9. **彼得·林奇的成功投资** - 成长股投资、PEG比率
10. **约翰·聂夫谈投资** - 低市盈率投资法、逆向投资

### 工具与参考
11. **交易术语大全** - 常用术语速查
12. **学习体系总览** - 完整学习路径规划

## 🎯 学习建议

1. **按顺序学习**：不要跳步，基础很重要
2. **做笔记**：每学完一章写总结
3. **做练习**：完成每个文档后的练习任务
4. **模拟交易**：先模拟，再实盘
5. **定期复盘**：每周回顾学习内容
