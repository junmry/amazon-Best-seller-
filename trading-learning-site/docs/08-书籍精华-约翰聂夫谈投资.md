## 一、作者介绍：约翰·聂夫

约翰·聂夫是全球著名的价值投资者，曾任温莎基金（Windsor Fund）经理。在1964-1995年的31年间，他管理的温莎基金年化收益率达到13.7%，远超同期标普500指数的10.6%。

**核心成就**：
- 管理温莎基金31年（1964-1995）
- 年化收益率13.7%（同期标普500为10.6%）
- 基金规模从7500万美元增长到110亿美元
- 以低市盈率投资法闻名

## 二、核心思想：低市盈率投资法

### 2.1 什么是低市盈率投资？

> **"在低市盈率时买入，在高市盈率时卖出。"**

**核心逻辑**：
- 市场往往过度悲观，导致好公司被低估
- 低市盈率 = 低风险 + 高回报潜力
- 耐心等待市场重新认识价值

### 2.2 低市盈率的优势

| 优势 | 说明 |
|------|------|
| **安全边际** | 估值低，下跌空间有限 |
| **高股息** | 低估值往往伴随高股息率 |
| **反弹潜力** | 估值修复带来超额收益 |
| **心理优势** | 买入成本低，心态更稳 |

### 2.3 低市盈率的陷阱

> **"不是所有低市盈率股票都值得买入。"**

**需要避免的情况**：
- 基本面恶化的公司
- 行业处于长期衰退
- 公司财务造假
- 周期性行业在高峰时

## 三、温莎基金的选股标准

### 3.1 七大选股要素

| 要素 | 标准 | 说明 |
|------|------|------|
| **1. 低市盈率** | 低于市场平均40-60% | 核心标准 |
| **2. 基本面成长率** | 超过7% | 确保有增长 |
| **3. 股息收益率** | 有保障 | 提供安全垫 |
| **4. 总回报率/市盈率** | 市盈率=总回报率的一半 | 估值与增长匹配 |
| **5. 周期性股票** | 需要低市盈率补偿 | 周期股特别标准 |
| **6. 成长行业中的稳健公司** | 行业+公司双优 | 质量要求 |
| **7. 强劲的基本面后盾** | 财务健康 | 风险控制 |

### 3.2 总回报率/市盈率关系

> **总回报率 = 盈利增长率 + 股息率**
> 
> **理想状态：市盈率 = 总回报率的一半**

**案例**：
- 公司A：增长率15%，股息率3%，总回报率18%
- 理想市盈率 = 18% ÷ 2 = 9倍
- 如果实际市盈率 = 7倍 → 低估 ✓
- 如果实际市盈率 = 15倍 → 高估 ✗

### 3.3 盈利增长率的要求

> **"增长率超过7%"**

**解读**：
- 7%是美国长期GDP增长率
- 超过7% = 跑赢经济
- 不需要高增长，稳定增长即可
- 避免负增长或零增长的公司

## 四、行业与周期

### 4.1 行业分类投资

| 类型 | 特征 | 策略 |
|------|------|------|
| **成长行业** | 高增长，高估值 | 寻找被错杀的机会 |
| **周期行业** | 业绩波动大 | 在行业低谷买入 |
| **成熟行业** | 增长稳定 | 关注低市盈率 |
| **衰退行业** | 负增长 | 一般回避 |

### 4.2 周期性行业的投资

> **"周期性股票需要低市盈率补偿。"**

**操作方法**：
- 在行业低谷时买入（市盈率看起来高）
- 在行业高峰时卖出（市盈率看起来低）
- 需要判断行业周期位置

**案例：花旗银行**
- 买入：从33美元跌到8美元
- 持有：耐心等待复苏
- 卖出：涨超8倍

### 4.3 行业轮动的观察

> **"资金在不同行业间流动。"**

**观察方法**：
- 哪些行业被过度追捧？（回避）
- 哪些行业被过度冷落？（关注）
- 行业基本面是否在改善？

## 五、逆向投资

### 5.1 什么是逆向投资？

> **"在市场的极端情绪中寻找机会。"**

**核心观点**：
- 市场往往过度反应
- 极端悲观 = 买入机会
- 极端乐观 = 卖出机会
- 独立思考，不随波逐流

### 5.2 逆向投资的原则

| 原则 | 说明 |
|------|------|
| **独立思考** | 不盲从市场共识 |
| **耐心等待** | 等待极端情绪出现 |
| **深入研究** | 确认基本面没有恶化 |
| **分散投资** | 降低单一标的风险 |
| **控制仓位** | 分批买入，降低成本 |

### 5.3 逆向投资的案例

**案例一：花旗银行**
- 背景：房地产危机，银行被抛售
- 买入：从33美元买到8美元
- 结果：涨超8倍

**案例二：大西洋富田石油**
- 背景：石油危机，石油股被抛售
- 买入：低市盈率
- 结果：大幅反弹

## 六、投资组合管理

### 6.1 分散投资

> **"不要把所有鸡蛋放在一个篮子里。"**

**温莎基金的分散**：
- 持有50-100只股票
- 覆盖多个行业
- 不同市值规模

### 6.2 持仓周期

> **"平均持股期：3年"**

**解读**：
- 价值投资需要时间
- 等待估值修复
- 避免频繁交易
- 长期持有优质公司

### 6.3 再平衡

> **"定期调整投资组合。"**

**方法**：
- 卖出涨过多的股票
- 买入被低估的股票
- 保持组合的平衡
- 每年进行一次

## 七、投资心理

### 7.1 克服从众心理

> **"市场一致看好时，要保持警惕。"**

**从众的危害**：
- 买入时估值过高
- 卖出时估值过低
- 错失真正机会
- 陷入群体非理性

### 7.2 保持耐心

> **"价值投资需要耐心等待。"**

**耐心的重要性**：
- 估值修复需要时间
- 市场认识价值需要时间
- 避免频繁交易
- 长期持有优质公司

### 7.3 控制情绪

> **"不要让情绪影响投资决策。"**

**情绪管理**：
- 恐慌时不盲目卖出
- 狂热时不盲目买入
- 保持理性分析
- 坚持投资纪律
