## 一、作者介绍：彼得·林奇

彼得·林奇是全球最著名的基金经理之一，曾任富达麦哲伦基金（Fidelity Magellan Fund）经理。在1977-1990年的13年间，他将麦哲伦基金从2000万美元管理到140亿美元，年化收益率高达29%，远超同期标普500指数的15.8%。

**核心成就**：
- 管理麦哲伦基金13年（1977-1990）
- 年化收益率29%（同期标普500为15.8%）
- 基金规模从2000万美元增长到140亿美元
- 投资超过15000只股票

## 二、核心思想：普通人也能战胜市场

### 2.1 投资与投机的区别

> **"投资是购买你了解的公司的股票，投机是购买你一无所知的公司的股票。"**

| 投资 | 投机 |
|------|------|
| 基于研究 | 基于消息 |
| 长期持有 | 短期炒作 |
| 关注基本面 | 关注价格波动 |
| 风险可控 | 风险不可控 |

### 2.2 散户的优势

> **"散户比机构有三大优势。"**

**优势一：没有时间压力**
- 不需要每季度排名
- 可以长期持有
- 不需要追逐热点

**优势二：投资范围更广**
- 可以投资小盘股
- 可以投资冷门股
- 不受规模限制

**优势三：更了解身边的公司**
- 日常消费的公司
- 工作的行业
- 生活的社区

### 2.3 投资的第一步：观察生活

> **"最好的股票就在你身边。"**

**发现方法**：
- 你经常购买的产品
- 你工作行业的公司
- 你孩子喜欢的品牌
- 社区里新开的热门店铺

**案例**：
-  Dunkin' Donuts：林奇在出差时发现
-  沃尔玛：林奇在调研时发现
-  苹果：林奇的孩子喜欢iPod

## 三、公司分类法

### 3.1 六大公司类型

| 类型 | 特征 | 例子 | 投资策略 |
|------|------|------|----------|
| **缓慢增长型** | 规模大，增长慢 | 公用事业 | 高股息 |
| **稳定增长型** | 稳定增长10-12% | 可口可乐 | 长期持有 |
| **快速增长型** | 增长20-25% | 小型成长股 | 重点投资 |
| **周期型** | 业绩随经济周期波动 | 汽车、航空 | 择时买卖 |
| **困境反转型** | 陷入困境，可能恢复 | 克莱斯勒 | 高风险高回报 |
| **隐蔽资产型** | 资产被低估 | 房地产、专利 | 深度研究 |

### 3.2 不同类型公司的投资要点

**缓慢增长型**：
- 关注股息率
- 稳定的现金流
- 防御性配置

**稳定增长型**：
- 长期持有
- 复利增长
- 适合定投

**快速增长型**：
- 重点投资对象
- 高增长潜力
- 需要深入研究

**周期型**：
- 择时很重要
- 在行业低谷买入
- 在行业高峰卖出

**困境反转型**：
- 高风险高回报
- 需要耐心等待
- 分散投资

**隐蔽资产型**：
- 深度研究
- 发现市场忽视的价值
- 长期持有

## 四、选股准则

### 4.1 13条选股准则

1. **公司名字听起来枯燥乏味**（甚至可笑更好）
   - 枯燥的名字 = 少人关注 = 低估机会

2. **公司业务枯燥乏味**
   - 无聊的业务 = 少人研究 = 低估机会

3. **公司业务令人厌恶**
   - 令人厌恶 = 少人投资 = 低估机会

4. **公司从母公司分拆出来**
   - 分拆后往往被低估
   - 管理层激励更强

5. **机构没有持股，分析师不追踪**
   - 少人关注 = 低估机会
   - 一旦发现，上涨空间大

6. **公司被谣言包围**
   - 谣言导致低估
   - 谣言澄清后上涨

7. **公司业务让人感到压抑**
   - 殡葬、垃圾处理等
   - 少人竞争，稳定盈利

8. **公司处于一个增长为零的行业**
   - 行业无增长 = 少人进入
   - 现有公司享受高份额

9. **公司有一个利基**
   - 护城河保护
   - 定价权

10. **人们要不断购买公司的产品**
    - 重复消费
    - 稳定收入

11. **公司是高技术产品的用户**
    - 技术提升效率
    - 成本下降

12. **公司内部人士在买入自家股票**
    - 内部人最了解公司
    - 买入 = 看好未来

13. **公司在回购股票**
    - 回购 = 看好未来
    - 提升每股收益

### 4.2 避开的股票

| 类型 | 原因 |
|------|------|
| 热门行业的热门股 | 估值过高，风险大 |
| 被吹捧成"下一个"的公司 | 模仿者很少成功 |
| 多元化恶化的公司 | 主业不精，副业亏损 |
| 依赖大客户的供应商 | 议价能力弱 |
| 名字花里胡哨的公司 | 炒作概念，无实质 |

## 五、财务分析要点

### 5.1 关键财务指标

| 指标 | 公式 | 优秀标准 |
|------|------|----------|
| **市盈率（P/E）** | 股价 ÷ 每股收益 | 低于行业平均 |
| **PEG比率** | P/E ÷ 盈利增长率 | < 1（越低越好） |
| **负债率** | 总负债 ÷ 总资产 | < 50% |
| **现金头寸** | 现金及等价物 | 越多越好 |
| **增长率** | 收入/利润增长率 | > 20% |

### 5.2 PEG比率（核心指标）

> **PEG = 市盈率 ÷ 盈利增长率**

**解读**：
- PEG < 1：低估，值得购买
- PEG = 1：合理估值
- PEG > 1：高估，谨慎购买

**案例**：
- 公司A：P/E=20，增长率=20%，PEG=1.0（合理）
- 公司B：P/E=15，增长率=30%，PEG=0.5（低估）✓
- 公司C：P/E=30，增长率=10%，PEG=3.0（高估）✗

### 5.3 财务报表关注点

**利润表**：
- 收入是否持续增长？
- 利润率是否稳定或提升？
- 费用控制是否良好？

**资产负债表**：
- 负债率是否合理？
- 现金是否充足？
- 存货和应收账款是否正常？

**现金流量表**：
- 经营现金流是否为正？
- 自由现金流是否充足？
- 资本支出是否合理？

## 六、买入与卖出时机

### 6.1 买入时机

| 时机 | 说明 |
|------|------|
| **股市崩盘或大跌时** | 优质公司被错杀 |
| **公司短期遇到困难时** | 不影响长期价值 |
| **公司被分拆或重组时** | 往往被低估 |
| **行业暂时不受欢迎时** | 优质公司被连带下跌 |
| **公司回购股票时** | 内部人看好 |

### 6.2 卖出时机

> **"卖掉赚钱的股票，留下亏损的股票，就像拔掉鲜花，浇灌野草。"**

**正确的卖出理由**：

| 理由 | 说明 |
|------|------|
| **基本面恶化** | 业务前景变差 |
| **估值过高** | PEG > 2 |
| **有更好的投资机会** | 机会成本 |
| **投资逻辑改变** | 最初买入的理由不再成立 |

**错误的卖出理由**：
- ❌ 股价涨了，想落袋为安
- ❌ 股价跌了，想止损
- ❌ 别人都在卖
- ❌ 想把钱拿出来花

### 6.3 奇衡DK的发展

> **"拔掉野草，浇灌鲜花"**
- 野草：亏损的股票，及时止损
- 鲜花：盈利的股票，让利润奔跑
- 加一句："有花堪摘只需摘，莫待无花空折枝"

## 七、投资组合管理

### 7.1 分散投资

> **"不要把所有鸡蛋放在一个篮子里。"**

**建议**：
- 持有3-10只股票
- 不同行业
- 不同类型（缓慢增长、稳定增长、快速增长）

### 7.2 持仓比例

| 类型 | 建议比例 | 说明 |
|------|----------|------|
| 快速增长型 | 30-50% | 核心仓位 |
| 稳定增长型 | 20-30% | 稳健配置 |
| 缓慢增长型 | 10-20% | 防御配置 |
| 周期型 | 10-20% | 择时配置 |

### 7.3 定期检查

**每季度检查**：
- 公司基本面是否变化？
- 估值是否合理？
- 是否有更好的机会？

**每年检查**：
- 投资组合整体表现
- 是否需要再平衡
- 投资目标是否变化

## 八、投资心理

### 8.1 克服恐惧与贪婪

> **"股市的下跌就像科罗拉多州的寒冬一样平常。"**

**恐惧时**：
- 股市大跌时，优质公司被错杀
- 这是买入机会，不是卖出理由
- 长期看，股市总是上涨的

**贪婪时**：
- 股市大涨时，估值往往过高
- 保持理性，不要盲目追高
- 记住：树不会长到天上去

### 8.2 耐心与坚持

> **"投资股票需要耐心，而投机者缺乏的正是耐心。"**

**要点**：
- 好公司需要时间来证明
- 不要期望一夜暴富
- 复利需要时间
- 长期持有优质公司

### 8.3 独立思考

> **"不要盲从专家，要相信自己的研究。"**

**原则**：
- 自己做研究
- 形成独立判断
- 对自己的投资负责
- 别人的意见仅供参考
