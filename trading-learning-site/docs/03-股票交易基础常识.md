## 一、股票是什么？

### 1.1 股票的定义

股票是股份公司发行的所有权凭证，是股份公司为筹集资金而发行给各个股东作为持股凭证并借以取得股息和红利的一种有价证券。

**简单理解**：买股票 = 成为公司的股东

### 1.2 股票的分类

| 分类标准 | 类型 | 说明 |
|----------|------|------|
| **市场** | A股 | 在中国大陆发行，人民币计价 |
| | B股 | 中国大陆发行，外币计价（已边缘化） |
| | H股 | 中国香港发行 |
| | 美股 | 美国市场发行 |
| **板块** | 主板 | 大型成熟企业（600/000开头） |
| | 中小板 | 中型企业（002开头，已合并） |
| | 创业板 | 成长型企业（300开头） |
| | 科创板 | 科技创新企业（688开头） |
| | 北交所 | 专精特新企业（8开头） |
| **性质** | 蓝筹股 | 大型优质企业 |
| | 成长股 | 高增长潜力企业 |
| | 周期股 | 业绩随经济周期波动 |
| | 概念股 | 有特定概念炒作 |

## 二、股票交易基本规则

### 2.1 交易时间

| 时段 | 时间 | 说明 |
|------|------|------|
| **早盘集合竞价** | 9:15-9:25 | 可下单，9:20前可撤单 |
| **上午连续竞价** | 9:30-11:30 | 正常交易 |
| **下午连续竞价** | 13:00-14:57 | 正常交易 |
| **尾盘集合竞价** | 14:57-15:00 | 收盘集合竞价 |
| **休市** | 周末+法定节假日 | 不交易 |

### 2.2 交易单位

| 项目 | 规则 | 示例 |
|------|------|------|
| **最小单位** | 1手 = 100股 | 买100股 = 1手 |
| **买入数量** | 必须是100股的整数倍 | 100股、200股、300股... |
| **卖出数量** | 可以是任意数量（不足100股需一次性卖出） | 150股、50股 |
| **价格最小变动** | 0.01元 | 10.00元 → 10.01元 |

### 2.3 交易制度

| 制度 | A股规则 | 说明 |
|------|---------|------|
| **T+1制度** | 当日买入，次日才能卖出 | 防止过度投机 |
| **涨跌停板** | 主板±10%，创业板/科创板±20% |  ST股±5% |
| **价格笼子** | 买入≤卖一价102%，卖出≥买一价98% | 防止异常报价 |

### 2.4 交易费用

| 费用类型 | 费率 | 说明 |
|----------|------|------|
| **佣金** | 0.025%-0.03%（最低5元） | 券商收取，可协商 |
| **印花税** | 0.05%（卖出时收取） | 国家税务 |
| **过户费** | 0.001% | 中国结算收取 |

**计算示例**：
- 买入10万元股票：佣金约25-30元
- 卖出10万元股票：佣金25-30元 + 印花税50元

## 三、开户流程

### 3.1 选择券商

| 考虑因素 | 建议 |
|----------|------|
| **佣金费率** | 选择低佣金（万2.5以下） |
| **交易软件** | 选择好用的APP |
| **服务质量** | 线下网点、客服响应 |
| **研究支持** | 研报质量、投顾服务 |

**推荐券商**：华泰证券、国泰君安、招商证券、东方财富等

### 3.2 开户步骤

1. **准备材料**：身份证、银行卡、智能手机
2. **下载券商APP**：如"涨乐财富通"、"君弘"等
3. **在线申请**：
   - 上传身份证照片
   - 视频认证
   - 绑定银行卡
   - 风险测评
4. **等待审核**：通常1个工作日内完成
5. **入金交易**：银证转账，开始交易

### 3.3 账户类型

| 账户类型 | 用途 | 数量限制 |
|----------|------|----------|
| **沪A账户** | 交易上海股票（600/688开头） | 每人最多3个 |
| **深A账户** | 交易深圳股票（000/300开头） | 每人最多3个 |
| **资金账户** | 资金存取、银证转账 | 1个 |
| **信用账户** | 融资融券（需额外开通） | 1个 |

## 五、投资风险提示

### 5.1 股市风险等级

> **重要提示**：股票投资有风险，入市需谨慎！

| 风险类型 | 说明 | 应对方法 |
|----------|------|----------|
| **市场风险** | 整体市场下跌 | 控制仓位、分散投资 |
| **个股风险** | 单个股票暴雷 | 深入研究、止损 |
| **流动性风险** | 无法及时卖出 | 选择成交活跃的股票 |
| **政策风险** | 政策变化影响 | 关注宏观政策 |
| **操作风险** | 自身操作失误 | 学习知识、模拟练习 |

### 5.2 新手常见错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| 满仓操作 | 无退路，风险极大 | 单只股票不超过30% |
| 追涨杀跌 | 高买低卖，持续亏损 | 制定计划，严格执行 |
| 频繁交易 | 手续费高，心态失衡 | 减少交易频率 |
| 不听止损 | 小亏变大亏 | 设定止损，坚决执行 |
| 盲目跟风 | 成为接盘侠 | 独立思考，研究后再买 |
| 借钱炒股 | 压力巨大，容易爆仓 | 只用闲钱投资 |

### 5.3 风险控制原则

1. **闲钱投资**：只用3年内不用的钱
2. **分散投资**：至少5只股票，不同行业
3. **控制仓位**：单只股票不超过30%
4. **设定止损**：单笔亏损不超过总资金的2%
5. **保持现金**：永远保留一部分现金

## 六、投资术语 glossary

| 术语 | 解释 |
|------|------|
| **牛市** | 市场整体上涨趋势 |
| **熊市** | 市场整体下跌趋势 |
| **多头** | 看涨、买入的一方 |
| **空头** | 看跌、卖出的一方 |
| **利好** | 有利于股价上涨的消息 |
| **利空** | 有利于股价下跌的消息 |
| **洗盘** | 主力震仓，清洗浮筹 |
| **出货** | 主力卖出股票 |
| **套牢** | 买入后股价下跌，无法盈利卖出 |
| **解套** | 套牢后股价回升，恢复盈利 |
| **割肉** | 亏损卖出 |
| **踏空** | 股价上涨，未能买入 |
| **追涨** | 股价上涨时买入 |
| **杀跌** | 股价下跌时卖出 |
| **抄底** | 在低位买入 |
| **逃顶** | 在高位卖出 |
| **盘整** | 股价在一定区间内波动 |
| **突破** | 股价突破盘整区间 |
| **回踩** | 突破后回调到支撑位置 |
| **补仓** | 下跌后加仓，摊低成本 |
| **满仓** | 所有资金都买入股票 |
| **空仓** | 没有持有任何股票 |
| **轻仓** | 少量资金持有股票 |
| **重仓** | 大量资金持有股票 |

## 七、新手入门建议

### 7.1 学习路径

| 阶段 | 时间 | 任务 |
|------|------|------|
| **阶段1** | 1-2周 | 学习基础知识（K线、指标、规则） |
| **阶段2** | 2-4周 | 模拟盘练习，熟悉交易软件 |
| **阶段3** | 1-3月 | 小资金实盘，积累经验 |
| **阶段4** | 持续 | 系统学习，建立交易体系 |

### 7.2 推荐起步资金

- **模拟盘**：无限额，充分练习
- **小资金实盘**：1-5万元，体验真实市场
- **正式投资**：建议10万元以上，分散配置

### 7.3 心态建设

1. **降低预期**：年化收益15%-20%已是优秀水平
2. **接受亏损**：亏损是交易的一部分
3. **保持耐心**：财不入急门
4. **持续学习**：市场永远在变化
5. **记录总结**：每笔交易都要复盘
