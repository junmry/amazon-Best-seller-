# 技术指标入门（完全小白版）

> 本文档面向：已了解K线基础，但对技术指标一无所知的新手

---

## 一、什么是技术指标？

### 1.1 定义
技术指标是通过数学公式对历史价格、成交量等数据进行计算，得出的用于辅助判断市场走势的数值或图形。

### 1.2 技术指标的分类

| 分类 | 作用 | 代表指标 |
|------|------|----------|
| **趋势指标** | 判断趋势方向和强度 | MA、MACD、BOLL |
| **震荡指标** | 判断超买超卖、转折点 | KDJ、RSI、WR |
| **成交量指标** | 分析量能变化 | VOL、OBV |
| **波动率指标** | 衡量价格波动幅度 | ATR、布林带 |

### 1.3 使用原则

> **重要提示**：技术指标是辅助工具，不是圣杯！

1. **多指标共振**：单一指标信号不可靠，需要多个指标验证
2. **趋势优先**：趋势指标 > 震荡指标
3. **周期配合**：大周期指标 > 小周期指标
4. **量价配合**：价格指标 + 成交量指标

## 二、移动平均线（MA）

### 2.1 什么是MA？

移动平均线（Moving Average，简称MA）是某段时间内收盘价的平均值，用于平滑价格波动，识别趋势方向。

**计算公式**：
```
MA5 = (今日收盘价 + 昨日收盘价 + ... + 前4日收盘价) ÷ 5
```

### 2.2 常用MA周期

| 周期 | 名称 | 用途 |
|------|------|------|
| MA5 | 5日线 | 超短期趋势 |
| MA10 | 10日线 | 短期趋势 |
| MA20 | 20日线 | 中期趋势（月线） |
| MA60 | 60日线 | 中长期趋势（季线） |
| MA120 | 120日线 | 长期趋势（半年线） |
| MA250 | 250日线 | 超长期趋势（年线） |

### 2.3 MA的三种形态

```
多头排列（上涨趋势）：
股价 > MA5 > MA10 > MA20 > MA60

空头排列（下跌趋势）：
股价 < MA5 < MA10 < MA20 < MA60

横盘整理（无趋势）：
MA5、MA10、MA20相互缠绕
```

### 2.4 MA的买卖信号

| 信号类型 | 具体形态 | 操作含义 |
|----------|----------|----------|
| **金叉** | 短期MA上穿长期MA | 买入信号 |
| **死叉** | 短期MA下穿长期MA | 卖出信号 |
| **支撑** | 股价回调到MA附近反弹 | 买入机会 |
| **压力** | 股价反弹到MA附近回落 | 卖出机会 |

### 2.5 奇衡DK的MA应用

> **年线法则**：250日线（年线）作为多空分界线
- 股价在年线上方：多头市场，以做多为主
- 股价在年线下方：空头市场，以观望或做空为主
- **UBW仓位管理**：年线上方/下方采用不同的仓位策略

## 三、MACD指标

### 3.1 什么是MACD？

MACD（Moving Average Convergence Divergence，指数平滑异同移动平均线）是判断趋势方向和强度的经典指标。

### 3.2 MACD的构成

```
MACD由三部分组成：

1. DIF线（快线）：12日EMA - 26日EMA
2. DEA线（慢线）：DIF的9日EMA
3. MACD柱（柱状线）：(DIF - DEA) × 2
```

**图形表示**：
```
        │
DIF线 ──┼─────── 快线
        │
DEA线 ──┼───     慢线
        │
0轴  ───┼───────── 多空分界线
        │
  红柱  │  绿柱
  ████  │  ████
  ████  │  ████
```

### 3.3 MACD的买卖信号

| 信号 | 形态 | 含义 |
|------|------|------|
| **金叉** | DIF上穿DEA，在0轴上方 | 强势买入信号 |
| **死叉** | DIF下穿DEA，在0轴下方 | 强势卖出信号 |
| **0轴上金叉** | DIF在0轴上方上穿DEA | 多头加仓信号 |
| **0轴下死叉** | DIF在0轴下方下穿DEA | 空头加仓信号 |
| **顶背离** | 价格新高，MACD未新高 | 顶部警示 |
| **底背离** | 价格新低，MACD未新低 | 底部警示 |

### 3.4 奇衡DK的三重滤网（第一重）

> **周线MACD定方向**
- 周线MACD在0轴上方：只考虑做多
- 周线MACD在0轴下方：只考虑做空或观望
- **核心原则**：顺势交易，不逆势操作

## 四、KDJ指标

### 4.1 什么是KDJ？

KDJ（随机指标）是判断超买超卖的震荡指标，对价格转折敏感。

### 4.2 KDJ的构成

```
KDJ由三条线组成：

K线（快线）：反映最新价格变化
D线（慢线）：K线的平滑
J线：3K - 2D（更敏感）
```

### 4.3 KDJ的数值区间

| 区间 | 数值范围 | 含义 |
|------|----------|------|
| **超买区** | K、D > 80 | 可能见顶，考虑卖出 |
| **超卖区** | K、D < 20 | 可能见底，考虑买入 |
| **徘徊区** | 20-80之间 | 趋势延续，持仓观望 |

### 4.4 KDJ的买卖信号

| 信号 | 形态 | 操作 |
|------|------|------|
| **金叉** | K线上穿D线，在20以下 | 买入信号 |
| **死叉** | K线下穿D线，在80以上 | 卖出信号 |
| **J线>100** | J线进入极端超买 | 强烈卖出信号 |
| **J线<0** | J线进入极端超卖 | 强烈买入信号 |

### 4.5 奇衡DK的三重滤网（第二重）

> **日线KDJ找买点**
- 周线MACD>0（多头市场）的前提下：
  - 日线KDJ金叉 + K<30：买入机会
  - 日线KDJ死叉 + K>70：卖出信号

## 五、威廉指标（WR）

### 5.1 什么是WR？

威廉指标（Williams %R）是拉瑞·威廉姆斯发明的震荡指标，用于判断超买超卖。

### 5.2 WR的计算与表示

```
公式：WR = (Hn - C) ÷ (Hn - Ln) × 100

Hn：n日最高价
Ln：n日最低价
C：当日收盘价
```

**数值范围**：0-100

### 5.3 传统用法 vs 奇衡用法

| 用法 | 买入信号 | 卖出信号 |
|------|----------|----------|
| **传统用法** | WR < 20（超买回调） | WR > 80（超卖反弹） |
| **奇衡用法** | WR上穿50 | WR下穿50 |

### 5.4 奇衡核心策略：波幅破位

> **威廉指标波幅破位策略**
- WR上穿50：做空信号
- WR下穿50：做多信号
- **核心逻辑**：突破50中轴线，代表多空力量转换

## 六、布林通道（BOLL）

### 6.1 什么是布林通道？

布林通道（Bollinger Bands）由三条线组成，用于衡量价格波动范围和趋势。

### 6.2 布林通道的构成

```
上轨（UB）= 中轨 + 2倍标准差
中轨（MB）= 20日移动平均线
下轨（LB）= 中轨 - 2倍标准差

标准差：衡量价格波动幅度
```

**图形表示**：
```
    上轨 ─────────────── 压力线
         
    中轨 ─────────────── 趋势线
         
    下轨 ─────────────── 支撑线
```

### 6.3 布林通道的用法

| 形态 | 含义 | 操作 |
|------|------|------|
| **开口（喇叭口）** | 波动率扩大，趋势启动 | 顺势跟进 |
| **缩口（布林带收窄）** | 波动率收敛，即将变盘 | 等待突破 |
| **股价触及上轨** | 短期超买 | 考虑减仓 |
| **股价触及下轨** | 短期超卖 | 考虑加仓 |
| **股价沿上轨运行** | 强势上涨 | 持仓待涨 |
| **股价沿下轨运行** | 强势下跌 | 空仓观望 |

### 6.4 奇衡核心：威廉时钟

> **波幅坍缩与扩张**
- **波幅坍缩**：布林带收窄，波动率降低 → 关注时机（倒计时）
- **波幅扩张**：布林带开口，波动率增加 → 顺势操作
- **核心定理**：波动坍缩之后必然扩大，扩大之后未必坍缩

## 七、成交量（VOL）

### 7.1 什么是成交量？

成交量是某段时间内股票成交的总手数（1手=100股），反映市场参与度。

### 7.2 量价关系基础

| 量价组合 | 形态 | 含义 |
|----------|------|------|
| **量增价涨** | 成交量放大，价格上涨 | 健康上涨，趋势确认 |
| **量缩价涨** | 成交量缩小，价格上涨 | 上涨乏力，可能回调 |
| **量增价跌** | 成交量放大，价格下跌 | 恐慌抛售，可能见底 |
| **量缩价跌** | 成交量缩小，价格下跌 | 无人接盘，继续观望 |
| **天量** | 成交量异常放大 | 转折点信号 |
| **地量** | 成交量极度萎缩 | 变盘前兆 |

### 7.3 量价配合原则

> **价格上涨需要成交量配合**
- 无量上涨 = 虚假上涨（诱多）
- 放量上涨 = 真实上涨（资金入场）
- 放量下跌 = 真实下跌（资金出逃）
- 缩量下跌 = 可能是洗盘

## 八、指标组合使用

### 8.1 奇衡DK的三重滤网系统

```
┌─────────────────────────────────────────────────────────┐
│  第一重滤网 - 市场潮汐（周线MACD）                        │
│  → 辨识长期趋势，决定做多还是做空                         │
│  → 周线MACD白线在0轴以上才考虑做多                        │
├─────────────────────────────────────────────────────────┤
│  第二重滤网 - 市场波浪（日线KDJ）                         │
│  → 辨识逆潮汐方向的波浪                                   │
│  → 周线上升趋势中，利用日线跌势寻找买进机会                │
│  → 日线KDJ金叉+K值跌破30为买入信号                        │
├─────────────────────────────────────────────────────────┤
│  第三重滤网 - 盘中突破（盘中突破）                        │
│  → 辨认顺着潮汐方向的涟漪                                 │
│  → 日线底分型停顿（盘中过前一日高点买入）                 │
│  → 止损位置：日线底分型低点                               │
└─────────────────────────────────────────────────────────┘
```

### 8.2 指标组合原则

1. **趋势指标 + 震荡指标**：MACD（趋势）+ KDJ（时机）
2. **价格指标 + 成交量指标**：K线形态 + VOL
3. **多周期验证**：周线 + 日线 + 小时线

## 九、新手学习路径

### 9.1 学习顺序建议

| 阶段 | 学习内容 | 时间 |
|------|----------|------|
| 第1周 | MA移动平均线 | 掌握金叉死叉、支撑压力 |
| 第2周 | MACD指标 | 掌握0轴、金叉死叉、背离 |
| 第3周 | KDJ指标 | 掌握超买超卖、金叉死叉 |
| 第4周 | 布林通道 | 掌握开口缩口、上下轨 |
| 第5周 | 威廉指标 | 掌握波幅破位策略 |
| 第6周 | 三重滤网 | 综合应用所有指标 |

### 9.2 练习任务

1. **每日任务**：观察5只股票的MA、MACD、KDJ状态
2. **每周任务**：找出3个金叉/死叉信号，跟踪后续走势
3. **每月任务**：用三重滤网系统筛选股票，记录成功率

## 十、常见误区与提醒

| 误区 | 正确做法 |
|------|----------|
| ❌ 只看一个指标 | ✅ 至少3个指标共振 |
| ❌ 频繁切换指标 | ✅ 精通几个核心指标 |
| ❌ 指标万能论 | ✅ 指标只是辅助工具 |
| ❌ 追求100%准确 | ✅ 接受概率，做好止损 |
| ❌ 忽视大盘环境 | ✅ 先看大盘，再选个股 |

---

> **核心原则**：技术指标是"望远镜"，不是"水晶球"。它们帮助我们看清趋势，但不能预测未来。
