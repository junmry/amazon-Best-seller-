# 交易术语大全（新手必备）

> 本文档汇总股票交易中的常用术语，方便快速查阅

---

## 一、市场基础术语

| 术语 | 解释 |
|------|------|
| **A股** | 在中国大陆发行，以人民币计价和交易的股票 |
| **B股** | 中国大陆发行，以外币（美元/港币）交易的股票 |
| **H股** | 在中国香港上市的内地企业股票 |
| **美股** | 在美国市场发行的股票 |
| **主板** | 大型成熟企业，代码600/000/001开头 |
| **创业板** | 成长型企业，代码300开头，涨跌幅20% |
| **科创板** | 科技创新企业，代码688开头，涨跌幅20% |
| **北交所** | 专精特新企业，代码8开头 |
| **新三板** | 全国中小企业股份转让系统 |
| **ST股** | 特别处理股票，存在退市风险，涨跌幅5% |
| ***ST股** | 退市风险警示股票 |
| **蓝筹股** | 大型优质企业股票（如茅台、平安） |
| **白马股** | 业绩优良、高成长、高回报的股票 |
| **黑马股** | 原本不被看好，突然大幅上涨的股票 |
| **概念股** | 有特定概念炒作的股票（如AI概念、新能源概念） |
| **次新股** | 上市一年内的股票 |
| **次新股** | 上市一年内的股票 |

## 二、交易术语

| 术语 | 解释 |
|------|------|
| **T+1** | 当日买入，次日才能卖出（A股交易制度） |
| **T+0** | 当日可买卖（期货、可转债等） |
| **做多** | 买入股票，期待上涨获利 |
| **做空** | 借入股票卖出，期待下跌后买回获利（A股融券） |
| **多头** | 看涨市场的一方 |
| **空头** | 看跌市场的一方 |
| **开仓** | 建立新头寸（买入或卖出） |
| **平仓** | 了结现有头寸（卖出或买回） |
| **持仓** | 当前持有的股票 |
| **空仓** | 没有持有任何股票 |
| **满仓** | 全部资金都买入股票 |
| **半仓** | 一半资金买入股票 |
| **轻仓** | 少量资金买入股票 |
| **重仓** | 大量资金买入某只股票 |
| **加仓** | 在原有基础上再买 |
| **减仓** | 卖出一部分持仓 |
| **补仓** | 下跌后买入，摊低成本 |
| **追涨** | 股价上涨时买入 |
| **杀跌** | 股价下跌时卖出 |
| **抄底** | 在低位买入 |
| **逃顶** | 在高位卖出 |
| **割肉** | 亏损卖出 |
| **套牢** | 买入后股价下跌，无法盈利卖出 |
| **解套** | 套牢后股价回升，恢复盈利 |
| **踏空** | 股价上涨，未能买入 |

## 三、价格相关术语

| 术语 | 解释 |
|------|------|
| **开盘价** | 当日第一笔成交价格 |
| **收盘价** | 当日最后一笔成交价格 |
| **最高价** | 当日最高成交价格 |
| **最低价** | 当日最低成交价格 |
| **最新价** | 当前最新成交价格 |
| **涨停** | 当日涨幅达到上限（主板10%，创业板/科创板20%） |
| **跌停** | 当日跌幅达到下限（主板10%，创业板/科创板20%） |
| **涨停板** | 达到涨停价格，通常买盘巨大 |
| **跌停板** | 达到跌停价格，通常卖盘巨大 |
| **一字板** | 开盘即涨停或跌停，K线呈"一"字形 |
| **炸板** | 涨停板被打开 |
| **封板** | 涨停后买盘巨大，维持涨停 |
| **破板** | 涨停板被大量卖单打开 |
| **高开** | 开盘价高于昨日收盘价 |
| **低开** | 开盘价低于昨日收盘价 |
| **平开** | 开盘价等于昨日收盘价 |
| **跳空** | 开盘价高于昨日最高价或低于昨日最低价 |
| **跳空高开** | 开盘价高于昨日最高价 |
| **跳空低开** | 开盘价低于昨日最低价 |
| **缺口** | 价格不连续的区域，通常会回补 |
| **回补缺口** | 价格回到缺口区域 |

## 四、成交量术语

| 术语 | 解释 |
|------|------|
| **成交量** | 某时段内成交的股票数量（手） |
| **成交额** | 某时段内成交的总金额 |
| **换手率** | 成交量/流通股本，反映活跃度 |
| **量比** | 当前成交量/过去5日平均成交量 |
| **放量** | 成交量明显放大 |
| **缩量** | 成交量明显缩小 |
| **天量** | 成交量异常放大，可能是转折点 |
| **地量** | 成交量极度萎缩，可能是变盘前兆 |
| **堆量** | 成交量持续放大，像堆积起来 |
| **无量上涨** | 上涨但成交量很小，可能不真实 |
| **量价齐升** | 价格上涨，成交量放大（健康） |
| **量价背离** | 价格与成交量走势不一致（警示） |

## 五、趋势术语

| 术语 | 解释 |
|------|------|
| **牛市** | 市场整体上涨趋势 |
| **熊市** | 市场整体下跌趋势 |
| **震荡市** | 市场在一定区间内波动 |
| **上升趋势** | 高点和低点不断抬高 |
| **下降趋势** | 高点和低点不断降低 |
| **横盘** | 价格在水平区间内波动 |
| **盘整** | 股价在一定区间内整理 |
| **突破** | 股价突破盘整区间 |
| **假突破** | 突破后迅速回落 |
| **回踩** | 突破后回调到支撑位置 |
| **反弹** | 下跌后暂时回升 |
| **反转** | 趋势方向改变 |
| **回调** | 上涨过程中的暂时下跌 |
| **反抽** | 下跌后的短暂回升 |
| **主升浪** | 主要上涨波段 |
| **主跌浪** | 主要下跌波段 |
| **阴跌** | 缓慢持续下跌 |
| **急跌** | 快速大幅下跌 |
| **拉升** | 快速上涨 |
| **砸盘** | 大单卖出，打压股价 |
| **护盘** | 大单买入，维持股价 |

## 六、财务术语

| 术语 | 解释 | 公式 |
|------|------|------|
| **市盈率(PE)** | 股价与每股收益的比率，衡量估值 | 股价 ÷ 每股收益 |
| **市净率(PB)** | 股价与每股净资产的比率 | 股价 ÷ 每股净资产 |
| **市销率(PS)** | 股价与每股销售额的比率 | 市值 ÷ 营业收入 |
| **PEG** | 市盈率与增长率的比率 | PE ÷ 盈利增长率 |
| **每股收益(EPS)** | 每股股票对应的净利润 | 净利润 ÷ 总股本 |
| **每股净资产(BPS)** | 每股股票对应的净资产 | 净资产 ÷ 总股本 |
| **股息率** | 分红与股价的比率 | 每股分红 ÷ 股价 |
| **ROE** | 净资产收益率，衡量盈利能力 | 净利润 ÷ 净资产 |
| **ROA** | 总资产收益率 | 净利润 ÷ 总资产 |
| **毛利率** | 毛利与收入的比率 | (收入-成本) ÷ 收入 |
| **净利率** | 净利润与收入的比率 | 净利润 ÷ 收入 |
| **负债率** | 负债与资产的比率 | 总负债 ÷ 总资产 |
| **现金流** | 企业实际收支的资金 | - |
| **净利润** | 扣除所有成本后的利润 | - |
| **营业收入** | 企业销售商品或服务的收入 | - |
| **总股本** | 公司发行的全部股票数量 | - |
| **流通股** | 可在二级市场交易的股票 | - |
| **市值** | 公司总价值 | 股价 × 总股本 |
| **流通市值** | 可交易部分的价值 | 股价 × 流通股 |

## 七、技术分析术语

| 术语 | 解释 |
|------|------|
| **K线** | 记录价格变动的图表 |
| **阳线** | 收盘价高于开盘价（国内红色） |
| **阴线** | 收盘价低于开盘价（国内绿色） |
| **实体** | K线中开盘价与收盘价之间的部分 |
| **影线** | K线中实体外的细线 |
| **上影线** | 实体上方的线 |
| **下影线** | 实体下方的线 |
| **均线(MA)** | 移动平均价格线 |
| **MACD** | 指数平滑异同移动平均线 |
| **KDJ** | 随机指标 |
| **RSI** | 相对强弱指标 |
| **BOLL** | 布林通道 |
| **WR** | 威廉指标 |
| **成交量(VOL)** | 成交数量指标 |
| **OBV** | 能量潮指标 |
| **金叉** | 短期均线上穿长期均线（买入信号） |
| **死叉** | 短期均线下穿长期均线（卖出信号） |
| **背离** | 价格与指标走势不一致 |
| **顶背离** | 价格新高，指标未新高（顶部警示） |
| **底背离** | 价格新低，指标未新低（底部警示） |
| **支撑位** | 股价下跌时可能反弹的位置 |
| **压力位** | 股价上涨时可能回落的位置 |
| **趋势线** | 连接价格高点或低点的直线 |
| **通道** | 两条平行趋势线形成的区间 |
| **形态** | 价格走势形成的特定图形 |
| **头肩顶** | 顶部反转形态 |
| **头肩底** | 底部反转形态 |
| **双顶(M头)** | 两次冲顶失败的顶部形态 |
| **双底(W底)** | 两次探底失败的底部形态 |
| **三角形** | 收敛整理形态 |
| **旗形** | 短暂整理后延续原趋势 |
| **楔形** | 收敛形态，可能反转 |

## 十、其他常用术语

| 术语 | 解释 |
|------|------|
| **IPO** | 首次公开发行（新股上市） |
| **定增** | 定向增发，向特定投资者发行股票 |
| **配股** | 向现有股东按比例发行新股 |
| **分红** | 公司将利润分给股东 |
| **送转股** | 送股和转增股本 |
| **除权** | 送转股后股价调整 |
| **除息** | 分红后股价调整 |
| **股权登记日** | 确定分红资格的日期 |
| **除权除息日** | 股价调整的日期 |
| **停牌** | 股票暂停交易 |
| **复牌** | 股票恢复交易 |
| **退市** | 股票从交易所摘牌 |
| **借壳上市** | 通过收购已上市公司实现上市 |
| **重组** | 公司资产、业务的重大调整 |
| **并购** | 收购与合并 |
| **股权激励** | 向员工授予股票或期权 |
| **回购** | 公司买回自己的股票 |
| **质押** | 将股票作为抵押物 |
| **解禁** | 限售股可以上市流通 |
| **大宗交易** | 大额股票交易 |
| **集合竞价** | 开盘前的撮合交易 |
| **连续竞价** | 正常交易时间的撮合方式 |
| **限价单** | 指定价格的订单 |
| **市价单** | 以当前最优价成交的订单 |
| **委托** | 下单的另一种说法 |
| **成交** | 订单被撮合完成 |
| **撤单** | 取消未成交的订单 |
| **滑点** | 实际成交价与预期的差异 |

---

> **提示**：术语只是工具，理解背后的含义和用法才是关键。建议结合实际交易逐步掌握。
