# K线基础入门（完全小白版）

> 本文档面向：从未接触过K线图表的纯新手

## 一、什么是K线？

### 1.1 K线的起源
K线（Candlestick Chart）起源于日本德川幕府时代（1603-1867年），由日本米市商人本间宗久发明，用于记录大米价格的波动。因其形状像蜡烛，英文称为Candlestick。

### 1.2 K线的基本构成

每根K线记录四个价格：
- **开盘价**：该时间段第一笔成交价格
- **收盘价**：该时间段最后一笔成交价格
- **最高价**：该时间段最高成交价格
- **最低价**：该时间段最低成交价格

```
        上影线（最高价到实体的线）
           │
    ┌──────┴──────┐
    │             │  ← 实体（开盘价和收盘价之间的部分）
    │             │
    └──────┬──────┘
           │
        下影线（最低价到实体的线）
```

### 1.3 阳线与阴线

| 类型 | 颜色（国内） | 颜色（国际） | 含义 | 形态特征 |
|------|-------------|-------------|------|----------|
| **阳线** | 红色 | 绿色 | 上涨 | 收盘价 > 开盘价 |
| **阴线** | 绿色 | 红色 | 下跌 | 收盘价 < 开盘价 |

**阳线示例**：
```
        │ ← 上影线（可能不存在）
    ┌───┤
    │ █ │ ← 实体（红色/空心）
    │ █ │
    └───┤
        │ ← 下影线（可能不存在）
```

**阴线示例**：
```
        │ ← 上影线
    ┌───┤
    │███│ ← 实体（绿色/实心）
    │███│
    └───┤
        │ ← 下影线
```

## 二、K线的时间周期

| 周期类型 | 每根K线代表 | 适用场景 |
|----------|------------|----------|
| 1分钟 | 1分钟的价格变动 | 超短线/日内交易 |
| 5分钟 | 5分钟的价格变动 | 短线交易 |
| 15分钟 | 15分钟的价格变动 | 短线交易 |
| 30分钟 | 30分钟的价格变动 | 短线/波段 |
| 60分钟（1小时） | 1小时的价格变动 | 波段交易 |
| 日线 | 1天的价格变动 | 中长线投资 |
| 周线 | 1周的价格变动 | 长线投资 |
| 月线 | 1个月的价格变动 | 超长线投资 |

**重要提示**：日线是最常用的周期，新手建议从日线开始学习。

## 三、单根K线的形态与含义

### 3.1 常见单根K线形态

| 形态名称 | 图形特征 | 市场含义 |
|----------|----------|----------|
| **大阳线** | 实体很长，上下影线短或没有 | 强烈上涨信号 |
| **大阴线** | 实体很长，上下影线短或没有 | 强烈下跌信号 |
| **小阳线** | 实体较短，上下影线也短 | 上涨力度弱 |
| **小阴线** | 实体较短，上下影线也短 | 下跌力度弱 |
| **十字星** | 开盘价≈收盘价，实体极短 | 多空力量均衡，可能反转 |
| **锤子线** | 下影线长，实体小，位于上方 | 底部反转信号 |
| **倒锤子** | 上影线长，实体小，位于下方 | 顶部反转信号 |
| **T字线** | 开盘价=收盘价=最高价，下影线长 | 下方支撑强 |
| **倒T字** | 开盘价=收盘价=最低价，上影线长 | 上方压力大 |
| **一字板** | 开盘价=收盘价=最高价=最低价 | 涨停或跌停 |

### 3.2 实体大小的意义

- **大实体**（实体长度占整根K线50%以上）：趋势强劲
- **小实体**（实体长度占整根K线25%以下）：趋势犹豫
- **实体位置**：
  - 阳线实体靠上：多头强势
  - 阴线实体靠下：空头强势

### 3.3 影线长度的意义

- **长上影线**：上方抛压重，可能见顶
- **长下影线**：下方支撑强，可能见底
- **无影线**：趋势非常明确

## 四、K线组合形态（基础篇）

### 4.1 两根K线组合

| 组合名称 | 形态特征 | 市场含义 |
|----------|----------|----------|
| **吞没形态** | 后一根K线完全包含前一根 | 趋势反转信号 |
| **乌云盖顶** | 大阳线后大阴线，阴线深入阳线 | 顶部反转 |
| **刺透形态** | 大阴线后大阳线，阳线深入阴线 | 底部反转 |
| **孕线** | 后一根K线完全在前一根实体内部 | 趋势可能暂停 |
| **平顶/平底** | 两根K线最高点/最低点相同 | 阻力/支撑明显 |

### 4.2 三根K线组合

| 组合名称 | 形态特征 | 市场含义 |
|----------|----------|----------|
| **早晨之星** | 阴线+十字星+阳线 | 底部反转信号 |
| **黄昏之星** | 阳线+十字星+阴线 | 顶部反转信号 |
| **红三兵** | 连续三根阳线，逐步升高 | 强势上涨信号 |
| **黑三鸦** | 连续三根阴线，逐步降低 | 强势下跌信号 |
| **多方炮** | 阳线+阴线+阳线（第三根创新高） | 上涨中继 |
| **空方炮** | 阴线+阳线+阴线（第三根创新低） | 下跌中继 |

## 五、K线阅读实战技巧

### 5.1 阅读K线图的步骤

1. **看趋势**：整体是上涨、下跌还是横盘？
2. **看位置**：当前K线在历史什么位置？
3. **看形态**：有没有特殊的K线形态？
4. **看量能**：配合成交量分析（后续课程）
5. **看周期**：多周期验证（日线+周线）

### 5.2 新手常见误区

| 误区 | 正确理解 |
|------|----------|
| ❌ 只看单根K线 | ✅ 要看K线组合和整体趋势 |
| ❌ 死记硬背形态 | ✅ 理解形态背后的多空博弈 |
| ❌ 追求100%准确 | ✅ K线是概率工具，不是预测工具 |
| ❌ 忽视时间周期 | ✅ 大周期比小周期更可靠 |

## 六、练习任务

### 练习1：K线识别（每日）
- 打开任意股票行情软件
- 观察当日K线形态
- 记录：颜色、实体大小、影线长度、特殊形态

### 练习2：K线日记（每周）
- 选择5只股票
- 记录一周的K线走势
- 标注重要的K线组合形态

### 练习3：形态识别（持续）
- 从历史走势中寻找：
  - 10个大阳线/大阴线
  - 5个十字星
  - 3个早晨之星/黄昏之星
